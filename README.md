# Awesome-Neural-Representation-for-Videos

A curated collection of papers, code, datasets, and other resources related to **Neural Representations for Videos (NeRV)**. This repository aims to provide researchers and practitioners with a comprehensive and up-to-date overview of the rapidly evolving NeRV field.

By [Qingyu Mao](https://scholar.google.com.hk/citations?hl=zh-CN&user=SWUKj1MAAAAJ).
